⏺ Here's the complete list of tools Claude Code can use, grouped by what they do. The ones you named — Read, Grep, Glob — are in the first group.

  File Reading & Searching

  - Read — Read file contents (with line numbers); also handles images, PDFs, Jupyter notebooks
  - Glob — Find files by name pattern (e.g. **/*.py recursive matching)
  - Grep — Search inside file contents using regex (ripgrep); respects .gitignore
  - LSP — Code intelligence via language servers (go-to-definition, find references, types)

  File Editing

  - Edit — Exact string replacement in an existing file
  - Write — Create a new file or overwrite an existing one
  - NotebookEdit — Edit Jupyter notebook cells

  Command Execution

  - Bash — Run shell commands
  - Monitor — Run a command in the background and stream its output

  Web & Network

  - WebFetch — Fetch a URL and extract info from it
  - WebSearch — Search the web

  Task & Scheduling

  - TaskCreate / TaskGet / TaskList / TaskUpdate / TaskStop — Manage a task list
  - CronCreate / CronDelete / CronList — Schedule recurring or one-shot tasks
  - ScheduleWakeup — Pace a self-running /loop

  Agents & Subagents

  - Agent — Spawn a subagent with its own context (what I just used for the claude-code-guide answers)
  - SendMessage — Message/resume an agent (experimental teams feature)

  Planning & Mode Control

  - EnterPlanMode / ExitPlanMode — Design an approach before coding, then get it approved

  Git & Worktrees

  - EnterWorktree / ExitWorktree — Work in an isolated git worktree

  MCP (Model Context Protocol)

  - ToolSearch — Find and load deferred/MCP tool schemas
  - ListMcpResourcesTool / ReadMcpResourceTool — List and read MCP resources
  - WaitForMcpServers — Wait for MCP servers still connecting

  Skills & Interaction

  - AskUserQuestion — Ask you a multiple-choice question
